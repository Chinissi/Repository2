### 1. Check Python version
-->

Run the following code to check the Python version that you have installed:  

```console title="Terminal command"
python --version
```

Great Expectations supports Python versions 3.7 through 3.11. If a Python 3 version number is not returned, run the following code:

```console title="Terminal command"
python3 --version
```

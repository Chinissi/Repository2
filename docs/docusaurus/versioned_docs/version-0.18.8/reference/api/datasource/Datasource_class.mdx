<dd><p>Returns a dictionary of data_asset_names that the specified dataconnector can provide.</p>
<p>Note that some data_connectors may not be
capable of describing specific named data assets, and some (such as
inferred_asset_data_connector) require the user to configure
data asset names.</p>
<p>Example return value:
<CodeBlock language="python">{`{
  data_connector_name: {
    names: [ data_asset_1, data_asset_2 … ]
  }
  …
}`}</CodeBlock></p>

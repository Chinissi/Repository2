<p>Base class for all SetColumnMapMetrics.</p>
<p>SetColumnMapMetric classes inheriting from SetColumnMapMetricProvider are ephemeral,
defined by their <cite>set</cite> attribute, and registered during the execution of their associated SetColumnMapExpectation.</p>
<p>Metric Registration Example:</p>
<p><CodeBlock language="python">{`map_metric = SetBasedColumnMapExpectation.register_metric(
    set_camel_name='SolfegeScale',
    set_=['do', 're', 'mi', 'fa', 'so', 'la', 'ti'],
)`}</CodeBlock></p>
<p>In some cases, subclasses of MetricProvider, such as SetColumnMapMetricProvider, will already
have correct values that may simply be inherited by Metric classes.</p>

<dd><p>Enum type, whose members depict the nature of return value of a metric implementation function (defined for a specified “ExecutionEngine” subclass) that is the final result (rather than a Callable for deferred execution).</p>
<p>The available types are:</p>
<ul class="simple">
<li><p><cite>VALUE</cite> – metric implementation function returns a value computed over a dataset represented by “Domain”         (e.g., a statistic on column row values).         This is the only value in use (others below have never been used and are thus deprecated).</p></li>
<li><p><cite>MAP_VALUES</cite> (never used and deprecated) – metric implementation function returns a mapping between every
“Domain” value and the result of a transformation of the corresponding “Domain” value.</p></li>
<li><p><cite>WINDOW_VALUES</cite> (never used and deprecated) – metric implementation function returns the result of applying a
specified windowing operation over “Domain” values.</p></li>
<li><p><cite>AGGREGATE_VALUE</cite> (never used and deprecated) – metric implementation function returns the result of applying a
specified aggregation operation to every “Domain” value.</p></li>
</ul>

<p>Enum type, whose members signify the data “Domain”, on which a metric can be computed.</p>
<p>A wide variety of “Domain” types can be defined with applicable metrics associated with their respective “Domain”
types.  The “Domain” types currently in use (<cite>TABLE</cite>, <cite>COLUMN</cite>, <cite>COLUMN_PAIR</cite>, and <cite>MULTICOLUMN</cite>) are declared here.</p>

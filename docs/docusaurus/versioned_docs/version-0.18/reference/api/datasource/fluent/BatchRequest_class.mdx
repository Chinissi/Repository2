<dd><p>A BatchRequest is the way to specify which data Great Expectations will validate.</p>
<p>A Batch Request is provided to a Data Asset in order to create one or more Batches.</p>
<dl class="field-list simple">
<dt class="field-odd">Parameters</dt>
<dd class="field-odd"><ul class="simple">
<li><p><strong>datasource_name</strong> – The name of the Datasource used to connect to the data.</p></li>
<li><p><strong>data_asset_name</strong> – The name of the Data Asset used to connect to the data.</p></li>
<li><p><strong>options</strong> – A dict that can be used to filter the batch groups associated with the Data Asset.
The dict structure depends on the asset type. The available keys for dict can be obtained by
calling DataAsset.batch_request_options.</p></li>
<li><p><strong>batch_slice</strong> – A python slice that can be used to filter the sorted batches by index.
e.g. <cite>batch_slice = “[-5:]”</cite> will request only the last 5 batches after the options filter is applied.</p></li>
</ul>
</dd>

<dd class="field-even"><p>BatchRequest</p>

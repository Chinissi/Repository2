<dd><p>Convenience method for testing yaml configs.</p>
<p>test_yaml_config is a convenience method for configuring the moving
parts of a Great Expectations deployment. It allows you to quickly
test out configs for system components, especially Datasources,
Checkpoints, and Stores.</p>
<p>For many deployments of Great Expectations, these components (plus
Expectations) are the only ones you’ll need.</p>
<p><cite>test_yaml_config</cite> is mainly intended for use within notebooks and tests.</p>
